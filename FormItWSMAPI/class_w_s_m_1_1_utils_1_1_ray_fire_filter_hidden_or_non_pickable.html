WSM::Utils::RayFireFilterHiddenOrNonPickable Class Reference

Description

Implementation of IRayFireFilterAndAbort that filters hidden and and non pickable objects Note that this implementation does not use a pick ray, as it is used by area selection where no pick ray exists.
