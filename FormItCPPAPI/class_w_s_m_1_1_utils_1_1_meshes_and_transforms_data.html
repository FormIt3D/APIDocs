WSM::Utils::MeshesAndTransformsData Class Reference

Description

Data that packages meshes and transforms together.

All the meshes should be drawn at all the transforms. So n meshes and m transforms means drawing m * n transformed meshes. Also if meshes do not have a material specified, use the material for the given transform.
